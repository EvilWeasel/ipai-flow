# IPAI Flow data

This directory holds the SQLite database (`ipai-flow.db`).
It is gitignored — your data stays local.

For Supabase demos, use `supabase/demo_seed.sql` to load a repeatable IPAI Flow dataset with precomputed summaries. If the app is deployed server-only with a service role key, `supabase/server_only_grants.sql` documents the safer grants variant.
